Here are all the output examples for each `agentpm` command:

## Project Setup Commands

### `agentpm init --epic epic-8.xml`
```xml
<init_result>
    <project_created>true</project_created>
    <config_file>.agentpm.json</config_file>
    <current_epic>epic-8.xml</current_epic>
</init_result>
```

### `agentpm config`
```xml
<config>
    <current_epic>epic-8.xml</current_epic>
    <project_name>MooCRM</project_name>
    <default_assignee>agent_claude</default_assignee>
</config>
```

### `agentpm switch epic-9.xml`
```xml
<switch_result>
    <previous_epic>epic-8.xml</previous_epic>
    <current_epic>epic-9.xml</current_epic>
    <updated>.agentpm.json</updated>
</switch_result>
```
